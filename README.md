# 📺 Projeto da trilha CSS da DIO - YouTube Clone - Layout com HTML e CSS

Projeto desenvolvido como desafio da **Digital Innovation One (DIO)** com o objetivo de praticar **estruturação de layout utilizando HTML5 e CSS3**, recriando uma interface inspirada na página inicial do YouTube.

## 📌 Objetivo

O projeto tem como finalidade reforçar conceitos fundamentais de **Front-End**, como:

- Estruturação semântica com HTML
- Organização de layout com **CSS Grid**
- Uso de **Flexbox** para alinhamento de elementos
- Criação de uma **barra lateral de navegação**
- Construção de **cards de vídeo**
- Organização visual semelhante a uma plataforma de streaming

## 🖥️ Preview do Projeto

Interface inspirada no YouTube com:

- Barra superior com ícones e logo
- Campo de busca
- Menu lateral de navegação
- Grid de vídeos com thumbnail, canal e informações

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Font Awesome** (ícones)
- **CSS Grid**
- **Flexbox**


## 🎯 Funcionalidades Implementadas

✔ Menu superior com ícones  
✔ Sidebar com navegação  
✔ Grid de vídeos organizado em colunas  
✔ Cards contendo:

- Thumbnail
- Avatar do canal
- Título do vídeo
- Número de visualizações
- Tempo de transmissão

✔ Layout organizado com **CSS Grid**

## 🎨 Conceitos de CSS Aplicados

- **Grid Layout** para organização dos vídeos
- **Flexbox** para alinhamento de elementos
- **Controle de rolagem da página**
- **Estilização de componentes**
- **Uso de ícones com FontAwesome**
